Browser comfort

A calmer frame for the open web.

Quartz keeps the browser surface quiet so pages feel spacious, readable, and fast without giving up native controls.

6 minute read Native toolbar WebKit rendering

Quartz starts from a simple idea: the browser should help you move through the web without making itself the loudest thing on screen. The window uses AppKit controls for navigation and lets WebKit do the work it was built to do.

The address field accepts full URLs, host names, local files, and plain search text. Back and forward state follows WebKit history, while reload and stop reflect the current page load.

For users who rely on extensions, Quartz can load local WebExtension packages on supported macOS versions. That keeps the core app small while leaving room for personal workflows.
